3. Describe how you solved a challenge or technical issue that you faced in a previous role (preferably in a previous support role). How did you determine that your solution was successful?


Since am currently working on a ticketing tool, there were complaints about a specific project that the ticket listing page was not accessible by users as the page was loading indefinitely. Below were the steps I took to resolve the concern -


  • Understood the issue and criticality of the situation from customer perspective.
  • Tried to reproduce the issue from my end.
  • Once I identified that the issue was reproducable, I immediately checked the application logs and browser logs to understand if there were any errors.
  • I enabled the general logs and slow logs next after discovering that there were no errors in application logs.
  • Through general logs, I identified a MySQL query that was taking infinite time to be executed. Took the same query and tried to execute in separate window which led me to the root cause.
  • The query didn't have a null condition check enabled due to which one of the tickets in listing page could not return any value leading to the indefinite loading.
  • I immediately modified the query by adding null condition and the issue was resolved from my end.
  • I cross-checked the same with customer post which I was confirmed that the issue has been resolved.
  • As a preventive measure, I instructed the platform teams to add a checklist during production upgrades to make sure all queries used in setup and finetuned and having null condition handled wherever applicable.
